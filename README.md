# vue-weather-app

Weather app made with Vue3. I used OpenWeatherMap API to get weather information.

## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```

### Lints and fixes files
```
npm run lint
```
